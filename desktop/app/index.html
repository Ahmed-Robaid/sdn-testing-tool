Software Defined Network Testing Tool

This is a tool for network performance measurement through OpenFlow.

This measurement tool is to generate customizable data streams with customizable throughputs. The tool also provides an user-friendly interface for throughput analytics. The approach is implemented as an OpenFlow APP and only generic OpenFlow switches are required. Compared to commercial solutions, like Spirent and IXIA, this tool is cheaper and more extensible. Compared to software-based solutions, like iperf/iperf3, this tool cost almost nothing in CPU loading and northbound interface. Moreover, this tool is designed as a virtual testing controller and can be easily deployed to cloud systems.

Enjoy this network performance measurement tool.
